Layar Terlalu Kecil

Lebar tampilan layar Anda terlalu kecil untuk menampilkan situs web ini dengan benar.

Apa yang bisa Anda lakukan?

  • Gunakan perangkat dengan lebar tampilan layar yang lebih besar.
  • Jika Anda menggunakan perangkat seluler, coba aktifkan Situs Desktop.
  • Putar perangkat Anda ke orientasi lanskap (horizontal).
  • Perkecil tampilan jika memungkinkan.

Gradient Mind

Browser Anda tidak mengaktifkan fitur JavaScript. Untuk menjaga fungsionalitas situs web, beberapa fitur yang bergantung pada JavaScript telah disembunyikan. Perlu diingat bahwa beberapa elemen penting situs web mungkin tidak berfungsi dengan baik tanpa JavaScript diaktifkan.

Jika Anda ragu dengan skrip yang ada, silakan lihat dokumentasi.
